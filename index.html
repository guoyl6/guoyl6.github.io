Pow(x,n)

  • 问题来源
  • 问题简介

    实现求幂函数

解题思路


求一个数的n次幂，并不需要进行n次乘法，如

    2的10次方，就是2的平方的5次方
    2的平方的5次方，就是2的平方的平方的2次方再乘以2
    以此类推

最后2的10次方只需进行4次乘法即可。

Code

#include <iostream>
using namespace std;

class Solution {
public:
    double myPow(double x, int n) {
        if (x == 0) {
            return 0;
        }
        if (n == 0) {
            return 1;
        }
        if (n < 0) {
            x = 1 / x;
            n = -n;
        }
        double t = x;
        while (n != 1) {
            x = n % 2 ? x * x * t : x * x;
            n /= 2;
        }
        return x;
    }
};
int main() {
    cout << Solution().myPow(2, 3) << endl;
    cout << Solution().myPow(2, -3) << endl;
    cout << Solution().myPow(2, 10) << endl;
    cout << Solution().myPow(2, 100) << endl;


    return 0;
}

Median of Two Sorted Arrays

  • 问题来源
  • 问题简介

    给定两个有序数组a[m], b[n]，找出中位数
    要求时间复杂度为O(log(m+n))

解题思路

  • 以 O(log(m+n)) 的时间复杂度将两个有序数组合成一个有序数组。
显然不行，想要让 m + n 个数有序，至少也要进行 m + n 次比较
  • 找出位于中间的数
这听起来像是废话。。其实，中位数就是位于中间的数，并不需要知道其两边数是否已正确的顺序排列。
